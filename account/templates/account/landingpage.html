About


This project aims to develop and design a fast, easy and secure online voting system.


This project was done by 4th year students under the course of Bachelor of Science in Information and Communication Technology, who are currently enrolled at Surigao del Norte State University.

SERVICES

Fast

This voting system will help us to minimize the voting time duration because aside providing the voters an online platform to vote, the system will automatically count the votes for each candidates.

Easy

The developers of this system made a simple user interface to make this web application user friendly.

Secure

This system is made by Django, a Python framework that has solid security. It contains clickjacking protection in the form of the X-Frame-Options middleware, which in a supporting browser can prevent a site from being rendered inside a frame.
